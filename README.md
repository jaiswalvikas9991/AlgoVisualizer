# AlgoVisualizer
A ReactJs based Web Application for visualization of Algorithms like Sorting, PathFinding etc.

> How to install?
  1. Clone the Repository
  2. Navigate to the directory
  3. Run "npm install"
  4. Run "npm start"
  5. Enjoy!!!
